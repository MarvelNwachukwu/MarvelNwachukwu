I write code from my mom's basement as part of **[BrainDAO's](https://braindao.org)** engineering team. I work across a bunch of their products: prediction markets and agent trading at **[IQ AI](https://app.iqai.com)**, AI agent tooling on **[adk-ts](https://github.com/IQAIcom/adk-ts)**, the blockchain encyclopedia at **[IQ.wiki](https://iq.wiki)**, and whatever else needs building.

2,500+ contributions this year. Still not moving out.

---

### What I'm building

- **Prediction Markets** · full-stack work on IQ AI's prediction platform, from order books to trading UI
- **Agent Trading** · building [IQ AI's agent trading platform](https://app.iqai.com) where AI agents trade prediction markets
- **AI Agent Tooling** · contributing to [adk-ts](https://github.com/IQAIcom/adk-ts), an AI agent development kit
- **Open Source**
  - [Recon](https://github.com/MarvelNwachukwu/Recon) · EVM wallet PnL terminal with FIFO cost-basis across 9 chains
  - [emit-watch](https://github.com/MarvelNwachukwu/emit-watch) · live decoded event feed for any smart contract
  - [use-custom-event-listener](https://github.com/MarvelNwachukwu/use-custom-event-listener) · React hook for custom DOM events
